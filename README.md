# Graph Theory Educational Website

An interactive, university-level educational web application dedicated to the comprehensive study of Graph Theory. The platform aims to bridge the gap between theoretical concepts and practical understanding through dynamic visualizations and intuitive explanations.

## Features

- **Comprehensive Content:** Spans 12 detailed sections, covering everything from fundamental concepts to advanced topics such as:
  - Minimum Spanning Trees (Kruskal's & Prim's algorithms)
  - Network Flow (Ford-Fulkerson method)
  - Graph Coloring
  - Connectivity
  - Eulerian and Hamiltonian problems
- **Interactive Visualizations:** Uses powerful rendering (like Mermaid.js) to bring graphs to life, allowing users to visually trace algorithms step-by-step.
- **Modern UI:** Designed with modern web technologies and vanilla CSS for an aesthetic, engaging, and responsive learning experience.

## Getting Started

This project is built using standard web technologies.

### Prerequisites

- [Node.js](https://nodejs.org/) (if using Vite/npm for local development)

### Installation & Running Locally

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the App directory:
   ```bash
   cd gtwebsite/app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
